Download Voxhora for Mac

Voxhora-Mac auto-updates silently from this site once installed. You install it once, and every future release lands the next time you open the app.

Download latest Mac release
Requires macOS 15.1 (Sequoia) or later. Works on Apple Silicon and Intel Macs.

Installation — three clicks

  1. Click the Download latest Mac release button above. It takes you to the GitHub Release page. Click the .dmg file to download.
  2. Open the downloaded Voxhora-Mac-X.Y.Z.dmg in your Downloads folder.
  3. Drag Voxhora-Mac.app onto the Applications folder shortcut in the window that appears.

To open it the first time, find Voxhora-Mac in your Applications folder and double-click it.

How auto-updates work

Voxhora-Mac checks this site for new versions every time you launch the app and every 24 hours after. When a new release is available, you'll see a small "Update available" dialog with a one-click Install button. That's the entire process — no reinstalling, no settings to configure, no terminal.

You can also force an update check any time: Voxhora menu → Check for Updates…

Uninstall

Drag Voxhora-Mac.app from the Applications folder to the Trash. Your billing data lives in iCloud and is not affected by removing the app — reinstalling Voxhora on the same Mac (or any other Apple device on the same Apple ID) restores everything.
